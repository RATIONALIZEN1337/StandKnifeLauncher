# StandKnife Launcher

<p align="center">
  <b>Утилита для запуска StandKnife с оптимизированными настройками</b>
</p>

---

## Описание

Маленькая программа которая делает игру с ПК в StandKnife более комфортной. Просто выбираешь настройки и жмешь кнопку - все применяется автоматически.

## Функции

- **DirectX**: DX11 как стандартный стабильный движок, DX12 как экспериментальный для новых видеокарт
- **Unity оптимизации**: Отключение VR модулей, прямой рендер графики, форсирование дискретной GPU, эксклюзивный полный экран
- **Windows оптимизация**: Отключение Fullscreen Optimizations и DPI Scaling для минимального input lag
- **Системный таймер**: Высокоточный таймер 0.5 мс для ровного Frame Time
- **Многопоточность GPU**: Принудительная загрузка драйвера видеокарты на 100%
- **FPS лимит**: Редактируемый ввод для ограничения кадров
- **Приоритет**: Высокий или RealTime приоритет для процесса игры
- **CPU Affinity**: Запуск только на физических ядрах — убирает фризы на SMT/HT
- **Игровой режим питания**: Автоматический план питания для максимума производительности без троттлинга

## Установка

1. Скачай архив из релизов
2. Распакуй куда удобно
3. Запусти StandKnifeLauncher.exe
4. Укажи путь к StandKnife.exe (кнопка с папкой)
5. Выбери настройки и жми **Играть с настройками**

---

## Шаблоны

| Шаблон | Описание |
|--------|----------|
| Макс. FPS | DX11 + высокий приоритет + отключенные логи |
| Без фризов | DX11 + только физические ядра CPU |
| Эксперимент | Скрытый DX12 для новых видеокарт |
| Стандарт | Обычный DX11 |

---

## Важно

- Все изменения можно откатить и применяются только после запуска на кнопку **Играть с настройками**
- Для игрового плана питания нужны права администратора
- Настройки сохраняются автоматически в %AppData%\StandKnifeLauncher
